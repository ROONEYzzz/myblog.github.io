hexo搭建步骤&避坑

一、构建本地hexo

npm install hexo -g

hexo init

npm install

hexo g

hexo s

打开本地浏览器即可查看

二、发布到github pages

1.绑定github：

git config --global user.name ""

git config --global user.email ""

2.添加密钥：

eval "$(ssh-agent -s)" 添加密钥到ssh-agent

ssh-add ~/.ssh/id_rsa 添加生成的SSH key到ssh-agent

3.在github上配置ssh：

新建一个new ssh key，将id_rsa.pub文件里的内容复制上去

ssh -T git@github.com 测试是否添加成功

4.配置_config.yml：

修改repo值： type,repository(值为github上ssh的路径),branch

修改url（github pages分配的url），root（项目名）（！！！项目名后缀最好为.github.io）

(不设置url和root，本地可以运行，线上运行无法加载css和js)

5.部署前安装扩展：

npm install hexo-deployer-git --save

6.部署发布：

hexo d -g
